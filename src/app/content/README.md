## 🎯 기존 계산기 (총 9개 계산기 + 콘텐츠 허브)

### 1. 급여 계산기 (`salary-calculator.html`) ⭐ 최고 인기
- **기능**:
  - 4대보험 자동 계산 (2025년 기준 - 국민연금 상한 265,500원)
  - 소득세 및 지방소득세 계산
  - 부양가족 공제 적용
  - 실수령액 산출
  - 연간 급여 요약
  - 급여 구성 비율 차트

### 2. 대출 계산기 (`loan-calculator.html`) ⭐ 매우 높음
- **기능**:
  - 원리금균등상환 / 원금균등상환 방식 지원
  - 월별 상환금액 계산
  - 총 이자 및 상환금 계산
  - 월별 상환 스케줄 테이블
  - 상환액 추이 그래프
  - 원금/이자 비율 파이 차트

### 3. 주택담보대출 계산기 (`mortgage-calculator.html`) ⭐ 높음
- **기능**:
  - LTV(담보인정비율) 계산
  - DTI(총부채상환비율) 계산
  - 대출 가능 금액 산출
  - 자기자본 및 대출금 구성 비율
  - 월 상환액 계산
  - 2025년 규제 반영

### 4. 퇴직금 계산기 (`severance-calculator.html`)
- **기능**:
  - 근속일수 자동 계산
  - 평균임금 기반 퇴직금 산출
  - 퇴직소득세 계산
  - IRP 절세 효과 안내
  - 근속연수별 퇴직금 테이블
  - 근로기준법 제34조 준수

### 5. 복리 이자 계산기 (`compound-interest-calculator.html`)
- **기능**:
  - 초기 투자금 + 월 적립금 복리 계산
  - 월/분기/반기/연 복리 주기 선택
  - 자산 증가 추이 그래프
  - 원금 대비 수익금 시각화
  - 연도별 자산 성장 분석

### 6. 연금 계산기 (`pension-calculator.html`)
- **기능**:
  - 은퇴 시점 예상 자산 계산
  - 국민연금 + 개인자산 통합 분석
  - 노후 생활비 시뮬레이션
  - 자산 증가 및 소진 추이 그래프
  - 2025년 평균 수령액 64만원 반영

### 7. 종합소득세 계산기 (`income-tax-calculator.html`) 🆕 HIGH CPC
- **기능**:
  - 프리랜서/자영업자 종합소득세 계산
  - 2025년 8단계 누진세율 (6~45%)
  - 사업소득, 기타소득, 임대소득 등 지원
  - 인적공제, 소득공제, 세액공제 반영
  - 5월 신고 대비 완벽 가이드
  - 소득세법 제55조 준수

### 8. 양도소득세 계산기 (`capital-gains-tax-calculator.html`) 🆕 HIGH CPC
- **기능**:
  - 부동산 양도소득세 정확 계산
  - 1가구1주택 비과세 요건 자동 판단
  - 다주택 중과세율 자동 적용 (조정대상지역)
  - 장기보유특별공제 (최대 80%)
  - 12억 초과분 과세 계산
  - 2025년 세율 및 규제 반영
  - 소득세법 제104조 준수

### 9. 벤처투자 세제혜택 계산기 (`venture-investment-calculator.html`) 🆕
- **기능**:
  - **소득공제형**: 개인투자조합, 벤처기업 직접투자, 크라우드펀딩
    - 3천만원 이하: 100% 공제
    - 3천~5천만원: 70% 공제
    - 5천만원 초과: 30% 공제
  - **세액공제형**: 벤처투자조합, 벤처투자펀드, 창업·벤처 전문 PEF
    - 투자금액의 10% 세액공제
    - 벤처투자펀드: 연간 한도 300만원
  - 공제한도 자동 계산 (소득공제형: 종합소득의 50%)
  - 예상 절세액 자동 산출 (소득세율 반영)
  - **3년 의무보유기간 검증**: 3년 미만 선택 시 계산 차단 및 경고
  - 실질 투자비용 계산
  - 조세특례제한법 제16조, 제16조의2, 제16조의3 / 벤처투자 촉진에 관한 법률 제2조 준수

### 10. 개인정보처리방침 (`privacy-policy.html`) 🆕
- **내용**:
  - 개인정보 미수집 명시
  - 익명 통계 데이터만 수집 (계산기 유형, 방문 시간, 브라우저 정보, 화면 크기)
  - 계산 데이터 클라이언트 처리 (서버 미전송)
  - 쿠키 미사용
  - Google AdSense 광고 관련 안내
  - 시각적으로 구분된 중요 내용 강조

### 11. 콘텐츠 허브 페이지 (`content/index.html`) 🆕
- **기능**:
  - 금융 정보 콘텐츠 통합 관리
  - 연봉 관련 정보 3개 콘텐츠 카드 형태로 표시
  - 모던한 그라디언트 디자인
