# ⚡ 5분 퀵 테스트 가이드

## 🚀 3단계로 시작하기

### 1️⃣ 설치 (2분)

```bash
cd nextjs-migration
npm install
```

### 2️⃣ 실행 (1초)

```bash
npm run dev
```

✅ 서버 실행 완료! → http://localhost:3000 자동 오픈

---
