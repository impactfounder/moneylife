## 🚀 빠른 시작

### 1. 프로젝트 설치 및 실행 (5분)

```bash
# 1. 터미널에서 프로젝트 폴더로 이동
cd nextjs-migration

# 2. 의존성 패키지 설치 (최초 1회만)
npm install

# 3. 개발 서버 실행
npm run dev
```

### 2. 브라우저에서 확인

```
✅ 서버가 정상 실행되면 자동으로 브라우저가 열립니다
✅ 또는 수동으로 http://localhost:3000 접속
```

## 🐛 문제 해결

### 문제 1: 서버가 실행되지 않음
```bash
# 해결 방법
1. Node.js 버전 확인 (18.0.0 이상 필요)
   node --version

2. 의존성 패키지 재설치
   rm -rf node_modules package-lock.json
   npm install

3. 포트가 사용 중인 경우
   npm run dev -- -p 3001
```

### 문제 2: 페이지가 로딩되지 않음
```bash
# 해결 방법
1. 브라우저 캐시 삭제 (Cmd+Shift+R / Ctrl+Shift+R)
2. 브라우저 콘솔(F12) 에러 확인
3. 터미널에서 에러 메시지 확인
```

### 문제 3: 계산 결과가 이상함
```bash
# 해결 방법
1. 입력값이 올바른지 확인
2. 브라우저 콘솔(F12)에서 에러 확인
3. 해당 계산기의 로직 파일 확인:
   - src/lib/*-calculator.ts
```

### 문제 4: TypeScript 에러 발생
```bash
# 해결 방법
1. TypeScript 의존성 확인
   npm install typescript @types/node @types/react @types/react-dom --save-dev

2. 타입 체크
   npm run build
```
